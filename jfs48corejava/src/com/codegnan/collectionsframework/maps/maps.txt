Map :
- It does not extend Collection interface
- Map interface represents the data in the form of Key and Value pairs. 
- Implementation classes of Map: 
   - HashMap
   - LinkedHashMap
   - TreeMap
   
 HashMap: 
 - HashMap is an implementation of Map interface
 - It uses put() method to add an entry in the HashMap. 
 -

 A hashmap will have key and value pairs
 Each Key and Value pair is known as Entry
 Set of such entries are known as EntrySet. 
 The set of keys in a hashmap is known as keyset.
 Group of values are known as values.
 
 The keys in HashMap must always be unique,
 But corresponding values can be repeated..

Method References:

Method references is a feature supported from java8

It uses a operator called double colon operator ::

Lmbda expression helps us to write inline
implementation of functional interface method. 

Method reference is a shorthand form of lambda expression

Method reference just calls the existing method of a class. 
For method reference we do not pass the arguments, they are 
passed automatically. 

2 types of method references: 
1. static method reference
- when a functional interface method matches a static method signature
   we can use a static method reference
   
Syntax:  
ClassName :: staticMethod

2. Instance method reference: 
Instance method reference is used when a functional interface method matches
with non-static method of a given object. 

Syntax:
objectRef :: instanceMethod
